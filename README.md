# Online Library Management System

A web application for managing a digital book catalog with search, sorting, reservation, and statistics. Data is stored in the browser using LocalStorage.

## Features
- Add books
- Delete books
- Search by title
- Sort alphabetically
- Reserve books
- Statistics dashboard
- Persistent storage with LocalStorage

## Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage

## Installation
1. Clone the repository
2. Open `catalogue.html`
